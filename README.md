
## Requisitos de desarrollo
-   Java JDK 17 o superior
-   JCalendar-1.4.jar
-   JUnit-4.13.2.jar , Hamcrest-2.2.jar
-   MySQL v8.0.31 - MySQL connector-8.0.31.jar

## Como Instalar

Descargar o clonar repositorio en directorio
